Demo Storyline

Use this page as a speaker script for presenting the project from data generation to operator decision support.

Demo Outcome

  • Cross-language system coherence
  • Validated packet integrity
  • Actionable alerts and reports
  • Operator-friendly control view

Step-by-Step Script

Step 1

Introduce the mission

Explain that AHMS simulates aerospace and healthcare monitoring where reliability and early anomaly detection are critical.

Step 2

Show telemetry generation

Present the C++ simulator creating packet data each second and attaching a checksum from the Assembly routine.

Step 3

Show backend verification

Describe Java Spring Boot recomputing checksum, rejecting corrupted packets, and persisting valid records.

Step 4

Show analysis intelligence

Walk through Python rules for high temperature, low oxygen, high vibration, and low battery alert generation.

Step 5

Show operator perspective

Show the C# control panel workflow: start monitoring, inspect latest values, review alerts, export report.

Step 6

Close with hardware logic

Explain the VHDL alert circuit and how it models a direct signal path for safety-critical conditions.
